這是一篇日記
　
同事要搬家了，地板重新拖過後擔心公婆不知道，所以讓先生提醒公婆進屋要換室內拖。
結果小女兒直接對婆婆說：「阿媽，媽媽說你會不換拖鞋！」
婆婆當場爆氣，說：「我知道要換拖鞋！」
丈夫出來打圓場說：「唉呀小孩子亂講話。」
小女兒說：「我才沒有亂講！」場面尷尬。
　
同事問我要怎麼告訴幼稚園的小女兒「有些話不能講」。
我想了想自己的例子，跟她說這個概念太難了，而且容易讓孩子質疑誠實的價值。
不過這件事應該還是能處理的，只是要先安撫女兒被說「亂講話」的情緒，並且要承認這是大人用詞不當。
接下來用提問的方式，詢問女兒說了實話後，阿媽的情緒如何？是不是生氣了？
再詢問女兒說實話為什麼讓阿媽不高興？
如果女兒想不出來，就把主角從阿媽換成女兒，讓她想像一下被說「你會不換拖鞋」會有什麼感覺？
（應該會產生　被質疑　之類的感覺）
再來與女兒核對，爸媽的本意是要提醒阿媽換拖鞋，
她是否覺得爸媽有在談話中質疑阿媽？

如果有　=>　與女兒核對是哪些語句讓女兒造成誤解　=>　父母修正用語、釐清語氣與語義
如果沒有　=>　與女兒討論她的表達方式可能會造成誤解　=>　（可示範提醒與質疑兩種口吻與女兒交談，詢問兩者帶來的情緒差異）　=>　帶女兒修正用語
　
同事問我需不需要告訴孩子「善意的謊言」，
其實這件事無關誠實與否。
孩子在接收父母的訊息，及表達個人的訊息，是否有100%的一致，是需要一再確認的；
如果父母能在這種情況下確認小孩的理解與感受，從而把握機會告訴他們如何妥善的表達自己，這對他們的語文能力及情緒掌控都會有很大的幫助。
（最後被讚美是親子專家，挖喔沒小孩的我能得到這個評語真是太意外了。教育真的很有趣啊！）
