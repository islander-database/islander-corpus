〈從紅皇后的跑步機上，下來〉
　
我非常喜歡愛麗絲夢遊仙境這個故事，

尤其喜歡紅皇后（Red Queen）這個角色。

紅皇后說過：

“Now, here, you see, it takes all the running you can do, to keep in the same place. If you want to get somewhere else, you must run at least twice as fast as that!”
（你要一直拚命地跑，才能保持在同一個位置。如果你想到別的地方，至少要跑得比現在快兩倍才行！）
　
在課堂上，我曾不止一次用這句話譏諷（x）激勵（ｏ）學生，

跟他們說：「我這個老師都全力在奔跑時，你們憑什麼不跑？」

其實這個世界也在用同樣的標準，逼著我跑。

我根本分不出來，到底我的哪一步是自願的，哪一步是被逼的；又或者早就糾纏在一起成了我體內的雙股DNA螺旋。

反正我本來就是被訓練良好，才會來當老師。

誰能跟我說，工廠裡生產的不透明水彩，能否畫出自然界真實的、變幻無窮的顏色？

別說不透明水彩了，即使是透明水彩、粉彩、油彩，各種可疊色的顏料，都沒有辦法還原這個世界的樣貌。

故宮的導覽老師說，印象派就是失敗了，不然後來也不會出現野獸派。

真矛盾，承認這個觀點，好像就等於承認我這樣被訓練好的人生，終究無法應付變化迅速、複雜且矛盾的社會。

儘管莫內的畫仍舊昂貴。但那又是有錢人的世界。

真不想當老師了。這句話可不是開學前隨便喊喊的而已。（好吧，其實還是喊喊而已；畢竟國文老師什麼不會，講話最會）

我只是想著新學期如何AI融入教學，就被演算法推薦了一堆課程，從心靈成長到寫作營再到寫coding；偶爾看到一篇心靈雞湯，明知雞湯沒用還是覺得自己需要般的喝下去，滑到最後發現還是在賣課程。

奇怪了，真的奇怪了。這樣中小學怎麼還會缺老師呢？

若我從紅皇后的跑步機下來，面前大概會是一面旋轉門吧。走進去旋轉門，另一頭還是跑步機的機率，嗯，很高啊。

也許是一排跑步機呢。不是只有一台。
